# Team photos

The root `README.md` references headshots under this directory (SVG initials by
default). To use real photos like
[lupantech/Eubiota](https://github.com/lupantech/Eubiota):

1. Add JPEG or PNG files (recommended **100×100** px, square crop), e.g.
   `fang_wu.jpg`, `weihao_xuan.jpg`, …
2. In `README.md`, change each `<img src="assets/team/….svg" …>` to the matching
   `.jpg` / `.png` path.
3. Update each person’s link (`<a href="…">`) to their homepage, Google
   Scholar, or GitHub profile.

You can remove the placeholder `*.svg` files once all JPEG/PNG assets are in
place.
