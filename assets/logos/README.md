# Institution logos

PNG/SVG files in this directory are shown in the root `README.md` institution
row (same pattern as [lupantech/Eubiota](https://github.com/lupantech/Eubiota)).

**Currently referenced:** `stanford.png`, `harvard.png`, `University of Tokyo.png`,
`Chinese University of Hong Kong.png`, `amazon.png`, and `thinking_bio_lab.svg`.

To add or reorder logos, edit the `<p align="center">` block near the top of
`README.md`. Filenames with spaces work best if you use `%20` in `img src` URLs
(e.g. `University%20of%20Tokyo.png`).
